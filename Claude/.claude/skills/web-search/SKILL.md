## 현재 날짜

## 검색 원칙
- **한국어 검색어만** 사용
- 검색어에 연도 포함: "AI 에이전트 트렌드 2026"
- 구체적인 검색어로 시작하고, 결과 부족 시 더 넓은 키워드로 재시도

## 신뢰할 수 있는 출처 (우선순위 순)
1. **전문 IT 매체**: ZDNet Korea, IT조선, 전자신문, 블로터, 테크크런치 코리아
2. **공식 기업/기관 블로그**: 삼성, LG, 카카오, 네이버, 정부 기관
3. **리서치/분석 기관**: 가트너 한국, IDC 한국, ETRI
4. **포털 뉴스**: 네이버 뉴스, 다음 뉴스 (주요 언론사 기사)

## 제외할 출처
- 광고/홍보성 콘텐츠 (URL에 utm_, 제목에 "후기", "추천" 등)
- 출처 불명 개인 블로그
- 1년 이상 된 자료 (트렌드 리서치의 경우)
- 내용이 너무 짧거나 개요만 있는 페이지

## WebFetch 사용 팁
- 각 URL에서 본문만 추출 (헤더, 광고 제외)
- 200자 미만은 유효 콘텐츠로 보지 않음
- 접속 실패 시 다음 URL로 이동
